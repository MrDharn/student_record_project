## 📌 Overview
This is a simple project built with **Node.js and Express** to demonstrate basic **CRUD operations** (Create, Read, Update, Delete) on student records.  
Instead of using a database, student data is stored in memory for simplicity.

---

## 🚀 Features
- **Create** – Add new student records via POST requests.
- **Read** – Retrieve all students or a single student by ID.
- **Update** – Modify existing student details.
- **Delete** – Remove student records.

---

## 🛠️ Tech Stack
- **Backend:** Node.js, Express
- **Data Storage:** In‑memory JavaScript array (no database)
